## Features
This website contains four pages, the Home, Conatct, About and Room page, all accessible through the navigation bar in the header.

* **Home page:** 
The home page comprises of the outer look of the page when the user open the website, it has a beautiful welcoming logo and picture of the hotel, at the navigation bar it has all the pages linked to each other. when scrow down it has the services we offer in the hotel, such as the Room, Bar and Events.

* **Room page:** 
This page contains  six beautiful picture of  rooms, the prices of each room, they are in flex display and well arrange.

* **Contact Us page:** 
The contact page has a form that helps the user to reach out to us and send queries or complaints, the forms contain user name, email, address and field to enter message.

* **About Us page:** 
The about us has information of the hotel, where to find us and basic services of  hotel offer.

## Deployment 

The live website has been deployed using GitHub Pages. Follow the steps below to deploy the site:

1. Log In to GitHub

   * If you don't have an account, sign up at GitHub.


2. Navigate to the Repository

   * Locate the repository for this project, portfolio1.(Exceptional hotel and apaprtment)

3. Access Repository Settings

    * Click on the Settings link in the repository navigation menu.

4. Configure GitHub Pages

    * In the left-hand side navigation bar, click on the Pages link.

5. Set the Source and Folder

   * In the Source section, select main from the drop-down menu for the branch.

6. Deploy the Site

    * Click the Save button to apply the changes and it will show green diploy successful.
